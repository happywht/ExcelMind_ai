## 🎯 项目概述

本项目旨在解决P0级架构问题：将文档生成从前端浏览器迁移到后端Node.js服务器，解决`docx-templates`库依赖Node.js专用模块导致的浏览器兼容性问题。

**核心问题**:
```
Error: vm.Script is not a constructor
Module "vm" has been externalized for browser compatibility
```

**解决方案**: 前后端分离架构，后端提供RESTful API，前端通过HTTP调用
