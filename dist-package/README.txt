ExcelMind AI 使用说明
====================

欢迎使用 ExcelMind AI！

## 系统要求

- Windows 10/11 (64位)
- 无需安装其他软件（已包含所有依赖）

## 快速开始

1. **启动应用**
   - 双击 `start.bat`
   - 等待约 5 秒，前端窗口会自动打开

2. **使用应用**
   - 后端服务器会在独立窗口中运行
   - 前端应用连接到后端提供 AI 功能

3. **停止应用**
   - 双击 `stop.bat`
   - 或直接关闭后端服务器窗口

## 文件说明

```
ExcelMind-AI/
```

## 常见问题

### Q: 提示"未找到 Node.js"怎么办？

A: 有两种解决方案：

**方案1：使用便携版 Node.js（推荐）**
1. 下载 Node.js 便携版：https://nodejs.org/dist/v20.11.0/node-v20.11.0-win-x64.zip
2. 解压到应用目录的 `nodejs` 文件夹
3. 确保 `nodejs\node.exe` 存在

**方案2：安装系统 Node.js**
1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本
3. 重启应用

### Q: 后端服务器启动失败怎么办？

A: 检查以下几点：
1. 端口 3001 是否被占用
   - 打开任务管理器，结束占用端口的程序
2. 查看后端窗口的错误信息
3. 确保 `.env` 文件配置正确

### Q: 前端无法连接后端怎么办？

A: 
1. 确保后端服务器窗口仍在运行
2. 检查后端窗口是否显示"✅ 后端服务器已启动"
3. 尝试访问 http://localhost:3001/health 检查后端状态

### Q: 如何配置 API 密钥？

A: 
1. 打开应用目录下的 `.env` 文件
2. 设置您的 API 密钥：
   ```
   ZHIPU_API_KEY=your_api_key_here
   ```
3. 重启应用
